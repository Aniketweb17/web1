# web1

A starter website scaffold built with HTML, CSS, and JavaScript.

## Files

- `index.html` — landing page layout
- `styles.css` — responsive styling
- `script.js` — interactive menu and form handling

## Run locally

Open `index.html` in your browser, or use a local web server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Customize

Edit `index.html`, `styles.css`, and `script.js` to make the website your own.
